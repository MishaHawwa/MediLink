## 🔍 Quick Reference

### Installation Commands
```cmd
# One-click install
install-all.bat

# Manual install
npm install
cd backend
pip install -r requirements.txt
```

### Run Commands
```cmd
# Backend (Terminal 1)
start-backend.bat
# OR
cd backend
python app.py

# Frontend (Terminal 2)
start-frontend.bat
# OR
npm start
```
