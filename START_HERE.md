## ⚡ Super Quick Start (5 Minutes)

### Step 1: Install Everything
**Double-click this file:** `install-all.bat`

This will automatically install all dependencies for both frontend and backend.

### Step 2: Start Backend Server
**Double-click this file:** `start-backend.bat`

Keep this window open! You should see:
```
* Running on http://127.0.0.1:5000
```

### Step 3: Start Frontend App
**Double-click this file:** `start-frontend.bat`

Your browser will automatically open to http://localhost:3000

## 🔧 Troubleshooting

### "npm is not recognized"
→ Install Node.js from https://nodejs.org/

### "python is not recognized"
→ Install Python from https://www.python.org/

### "Cannot connect to server"
→ Make sure backend is running (start-backend.bat)

### Map not loading
→ Check your internet connection (map tiles need internet)

- **AI/ML:** Random Forest (scikit-learn)
- **Real-time:** WebSocket
